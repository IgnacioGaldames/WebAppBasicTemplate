Download this project as a .zip file Download this project as a tar.gz file

Web App Template fo everyday use

This contains the basics for a ready-to-use web app it includes the meta-names and links for:

Animate: For a quick animation of you web app

Web app on Meta-names for an IOS web app capability

Opera Speed Dial Favicon Just in case

Standard Favicon Multiple favicon support

Web app icon IOS standard icon support

Web app splash screen Multiple app splash screens supported

Add 2 Home Support for Add to home feature

Web App Images in Illustrator and EPS format Original images in Illustrator and EPS format in ready-to-design templates
